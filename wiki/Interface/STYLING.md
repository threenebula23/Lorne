# Стилизация и Темы (STYLING)

TCA использует систему стилей, очень похожую на веб-технологии. Если вы умеете писать на CSS, вы легко разберетесь со стилизацией TCA.

## Файл стилей: `tui_app.tcss`
Это основной файл, где описан внешний вид всех элементов.
- **Селекторы**: Работают как в CSS (`#id`, `.class`, `WidgetName`).
- **Свойства**: `background`, `color`, `border`, `padding`, `margin`, `width`, `height`.
- **Flexbox**: Используются `1fr` для резиновых размеров (например, центральная колонка занимает всё свободное место).

Пример из кода:
```css
#col-left {
    width: 28;
    background: $surface;
    border-right: vline $panel-border;
}
```

## Переменные и Темы
В TCA есть встроенная поддержка тем (`Interface/themes.py`). 
- Переменные типа `$accent`, `$surface`, `$text` меняются в зависимости от выбранной темы.
- Чтобы добавить свою тему, создайте новый словарь в `Interface/themes.py` и зарегистрируйте его в списке доступных тем.

## Размер интерфейса (плотность)
На корневое приложение вешаются классы **`density-compact`**, **`density-normal`** или **`density-spacious`** (из настроек → `ui_prefs` → при старте `TCAApp.on_mount` и при смене в UI). В `tui_app.tcss` заданы отступы и высоты вкладок, кнопок чата, поля ввода и т.д. Добавляя новые виджеты в IDE, дублируйте нужные правила под этими селекторами, если размер должен масштабироваться вместе с остальным UI.

## Акцентные цвета
Пользователь может менять основной (акцентный) цвет интерфейса в настройках. Этот цвет передается в виджеты через метод `_ui_colors()`. Используйте его, если хотите, чтобы ваш новый элемент подстраивался под настройки пользователя.

---

## Как быстро изменить дизайн?
1. Запустите TCA.
2. Откройте `Interface/tui_app.tcss`.
3. Измените, например, фоновый цвет `#main`.
4. Если вы в режиме `--dev`, изменения применятся мгновенно. Если нет — перезапустите приложение.

## Иконки и Символы
Для иконок в TCA используются стандартные символы **Unicode** и шрифты типа **Nerd Fonts** (если они установлены у пользователя). Старайтесь использовать универсальные символы (⚙️, 📁, 🤖), которые отображаются во всех современных терминалах.
