# Документация инструмента: context7_tool.py (Библиотечная документация)

Этот инструмент предоставляет агенту прямой доступ к актуальной документации популярных библиотек и фреймворков (React, FastAPI, LangChain и др.) через сервис Context7.

## Для всех: Что это делает?
Это "библиотекарь" для разработчика. Вместо того чтобы искать документацию вручную на разных сайтах, агент может:
- **Найти библиотеку**: Узнать, есть ли в базе документация по конкретной технологии.
- **Получить справку**: Задать вопрос ("Как использовать хуки в React?") и получить точный ответ с примерами кода из официальных источников.

Если у вас нет специального ключа Context7, инструмент просто переключится на обычный поиск в интернете.

---

## Для разработчиков: Разбор кода (подробно)

### Интеграция с API (Строки 19–53)
- Использует REST API версии 2 от Context7.
- **`_c7_request`**: Выполняет HTTP-запрос с авторизацией через Bearer токен. Результаты кэшируются в памяти (`_CACHE`), чтобы не делать повторных запросов в рамках одной сессии.

### Запасной вариант (Строки 56–80)
- **`_ddgs_fallback`**: Если ключ API не задан (`CONTEXT7_API_KEY`), инструмент вызывает поиск через DuckDuckGo (`ddgs`), добавляя к запросу слово "documentation". Это гарантирует, что агент всё равно получит информацию, хотя и менее структурированную.

### Основные инструменты

#### `resolve_library` (Строки 84–107)
- Ищет соответствие между названием (например, "react") и внутренним ID сервиса (например, "/reactjs/react.dev").
- Возвращает краткое описание и "Trust Score" (оценку доверия к источнику).

#### `get_library_docs` (Строки 110–145)
- Основная "рабочая лошадка". Принимает ID библиотеки и текстовый запрос.
- Возвращает `context` — очищенный текст документации, оптимизированный для понимания нейросетью.
- Ограничивает ответ по токенам (`max_tokens`), чтобы не "забивать" память агента.

#### `get_documentation` (Строки 149–173)
- Комбинированный инструмент "всё в одном". Сначала пытается найти ID библиотеки, а затем сразу запрашивает документацию. Если что-то не получается — уходит в поиск.

---

## Особенности
1. **Точность**: Context7 дает более точные ответы, чем обычный поиск, так как база индексируется специально для AI.
2. **Бесплатный ключ**: Вы можете получить API-ключ на сайте context7.com и добавить его в файл `.env` как `CONTEXT7_API_KEY`.

---

## Примеры использования

**Узнать про конкретную функцию:**
```python
get_documentation(query="как работает useEffect", library="react")
```

**Поиск библиотеки:**
```python
resolve_library(library_name="fastapi")
```
