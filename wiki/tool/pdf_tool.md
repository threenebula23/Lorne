# Документация инструмента: pdf_tool.py (Создание документов)

> **Для вызова у модели:** **`create_pdf`** (и связанные PDF-тулы из реестра) не объединены в один диспетчер. См. [COMPACT_TOOLS.md](../COMPACT_TOOLS.md).

Этот инструмент позволяет агенту генерировать отчеты, спецификации или документацию в формате PDF.

## Для всех: Что это делает?
Это "принтер" агента.
- Вы можете попросить: "Составь отчет о проделанной работе и сохрани в PDF".
- Агент берет текст, оформляет его и создает файл `.pdf`, который можно открыть в любой программе для чтения документов.

Если на вашем компьютере не установлена библиотека для создания PDF, инструмент автоматически создаст обычный текстовый файл `.txt` с тем же содержанием, чтобы информация не потерялась.

---

## Для разработчиков: Разбор кода (подробно)

### Реализация создания (Строки 15–38)
- **Библиотека**: Для генерации используется `reportlab`.
- **Логика**:
  1. Создается "холст" (`canvas`) размера A4.
  2. Печатается заголовок крупным шрифтом (`Helvetica`, 16).
  3. Текст тела письма разбивается на строки.
  4. Код следит за тем, чтобы текст не выходил за края страницы. Если страница заканчивается (`y < 2 cm`), вызывается `c.showPage()` — это создает новую страницу в документе.
- **Обработка ошибок (Заглушка)**: Если библиотека `reportlab` не установлена (ImportError), срабатывает блок `except`. Вместо падения, программа создает файл `.txt` и возвращает статус `created_as_txt`, вежливо напоминая о необходимости установки библиотеки (`pip install reportlab`).

### Основная функция `create_pdf` (Строки 42–47)
- Проверяет расширение файла. Если пользователь или агент указали путь без `.pdf`, инструмент добавит его автоматически.
- Вызывает внутреннюю реализацию создания.

---

## Особенности
- **Helvetica**: Используется стандартный шрифт. Важно помнить, что стандартные шрифты PDF могут плохо поддерживать кириллицу (русский язык) без дополнительной настройки шрифтов `.ttf`. (В текущей версии реализована база).
- **Авто-пагинация**: Инструмент базово умеет переносить текст на новые страницы, если текст очень длинный.

---

## Примеры использования

**Создание простого отчета:**
```python
create_pdf(
    filepath="report.pdf", 
    title="Отчет по проекту", 
    body="1. Исправлены баги\n2. Добавлен новый функционал"
)
```
